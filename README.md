## Features

- Fireplace video loop
- Christmas music playlist
- F1 quotes at random intervals (5-10 minutes)
- Settings menu
- PWA support (no offline caching for songs cause SW is weird)
- Made as a [Giftbox](https://giftbox.hackclub.com/) gift

## Quick Start

1. Clone the repository

   ```bash
   git clone https://github.com/BananaJeanss/fireplace-paddock.git
   cd fireplace-paddock
   ```

2. Install dependencies

   ```bash
   npm i
   ```

3. Run the development server

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `https://localhost:5173` (mkcert is used for HTTPS, cause PWA requires it)
